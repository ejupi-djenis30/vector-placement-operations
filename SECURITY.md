# Privacy and security

The application is static and contains synthetic fixtures only. It has no authentication, backend, cookies, analytics or network API. User changes are written to local storage under the `vector-placement-demo` key and can be cleared from the interface.

Please report vulnerabilities through GitHub's private security advisory feature. Include clear reproduction steps and the affected browser or runtime. Do not include real student, company or staff records in a report.
